# Alpha Social Game Master

## One-Liner

Alpha turns a group chat into a living social game that creates rituals, quests, memories, plans, and playful momentum.

## Core Intent

> "Make this group feel more connected, playful, and alive."

## Why This MVP Probe

The Social Game Master is a narrow surface for the broader Alpha thesis.

It proves that:

- intent can become an ongoing experience
- agents can coordinate multiple humans
- generated media and game mechanics can make ordinary life feel richer
- memory creates compounding value
- the product can live where people already are

It is not a productivity tool with game mechanics pasted on top. It is a system that uses game design to coordinate real social energy.

This is not the whole Alpha concept. It is one possible probe into the paradigm: a way to test whether intent can become an adaptive lived experience with memory, permissions, generated interfaces, and human feedback.

## User Experience

A user creates an Alpha space for a group.

Alpha asks a few high-leverage questions:

- Who is this group?
- What should feel different in a month?
- What is the group's energy level?
- What topics, jokes, or boundaries matter?
- How often should Alpha intervene?

Alpha then proposes a season:

- theme
- group goal
- roles
- weekly rhythm
- first mission
- optional real-world plan

Every week, Alpha:

- creates a mission
- adapts to participation
- proposes plans
- produces recaps
- evolves the lore
- remembers what mattered
- makes the next interaction easier

## Example Season

Group:

> Five friends who used to see each other often but now mostly send memes.

Intent:

> "Make us feel like a real crew again without making it cringe."

Alpha creates:

- Season: "The Comeback Department"
- Roles: Instigator, Archivist, Wildcard, Navigator, Closer
- Week 1 mission: each person sends one photo from a forgotten shared memory
- Week 2 mission: Alpha creates a mini ranking game from the memories
- Week 3 mission: schedule a low-friction 90-minute meetup
- Week 4 mission: produce a shared artifact: zine, playlist, mockumentary trailer, or private page

Success is not app engagement.

Success is:

- people laugh
- people show up
- people remember why they like each other
- the group does something it would not have done otherwise

## First Version

The first version can be simple:

- Web app for creating a group space.
- Chat-like interface for the organizer.
- Generated weekly mission cards.
- A group invite link.
- Shared timeline.
- Memory capture.
- Recap generation.
- Manual integrations at first: copy/paste to WhatsApp, Discord, iMessage, Slack.

No need to overbuild automation early. The key is taste, rhythm, and proof of aliveness.

## Feature Set

### Must Have

- Intent intake
- Group profile
- Season generator
- Weekly mission generator
- Shared mission page
- Recap creator
- Memory log
- Basic feedback loop

### Should Have

- Role assignment
- Tone controls
- Participation tracking
- Calendar suggestion
- Image/media prompts
- Private group archive

### Later

- Discord/WhatsApp/Slack integrations
- AI NPCs
- Voice mode
- Payment for experience packs
- Creator-made seasons
- Local venue/service coordination
- Real-world prop kits

## Product Principle

The system should never feel like homework.

It should feel like the group has a strange, funny, emotionally intelligent producer who knows when to nudge and when to disappear.
