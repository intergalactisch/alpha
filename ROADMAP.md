# Roadmap

This is a conceptual roadmap, not a fixed product plan.

Alpha is still being discovered. The roadmap exists to keep direction visible while preserving room for change, uncertainty, and better language.

## Current Center

**Shape Life**

Alpha helps people shape life their way, with care.

The current working definition:

> Alpha is accountable mediation between what people mean and what their world around them can become.

## Phase 0: Conceptual Ground

Goal: make the conceptual foundation clear enough for others to read, challenge, and extend.

Current work:

- refine what Alpha is and is not
- keep the concept open without making it vague
- clarify trust, safety, children, memory, culture, creativity, communication, and economic boundaries
- collect language that feels warm, human, and precise
- map the old paradigm Alpha responds to

Possible outputs:

- stronger manifesto
- conceptual map
- public glossary
- decision log
- high-level Alpha kernel

## Phase 1: Formal Shape

Goal: create precise enough specs and protocols to make Alpha buildable.

Possible work:

- intent/context specs
- memory and consent specs
- process object and queue specs
- artifact specs
- agent-to-agent and human-to-agent protocol sketches
- trust, transparency, and reversibility protocols
- media-agnostic interaction model

Possible outputs:

- `docs/specs/*`
- `docs/protocols/*`
- conceptual diagrams
- safety and transparency requirements

## Phase 2: Explorations

Goal: try concrete technical forms without pretending they are the final form of Alpha.

Possible work:

- personal intent space prototype
- process/queue visibility prototype
- dynamic contextual clustering prototype
- memory consent prototype
- media-agnostic option rendering prototype
- agent orchestration experiments

Possible outputs:

- technical tryouts
- implementation notes
- demos
- failure reports
- revised specs

## Phase 3: First Living Loop

Goal: build a first Alpha-shaped loop that can understand intent, agree on an outcome, coordinate work, and return a cared-for result.

Possible work:

- a user starts from text, voice, image, document, or situation
- Alpha clarifies intent only when needed
- Alpha proposes options in the richest available medium
- Alpha tracks work without hiding uncertainty
- Alpha produces an outcome the user can accept, correct, reshape, or delete

Possible outputs:

- first integrated prototype
- protocol-backed artifacts
- transparent process view
- documented learnings

## Phase 4: Public Paradigm Proof

Goal: show that Alpha can be more than software generation.

Possible work:

- life orchestration
- creative co-creation
- trusted communication
- memory and continuity
- social and cultural interaction
- ethical prosperity
- entertainment and shared joy

Possible outputs:

- public demo
- reference architecture
- community contribution model
- Alpha-shaped ecosystem experiments
