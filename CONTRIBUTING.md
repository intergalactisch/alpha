# Contributing To Alpha

Alpha is in a highly conceptual phase.

Contributions are welcome when they help the concept become clearer, more humane, more possible, or more carefully buildable.

## How To Contribute

Useful contributions can include:

- conceptual critique
- better language
- research notes
- diagrams
- protocol sketches
- spec proposals
- technical tryouts
- safety concerns
- examples from everyday life
- examples from culture, entertainment, business, education, creativity, memory, or community

## What To Protect

Alpha should not be pushed too quickly into a narrow product form.

Please protect:

- human agency
- consent
- transparency
- privacy
- child safety
- cultural dignity
- creative dignity
- optional reciprocity
- non-coercive communication
- ecological and cost awareness
- the right to uncertainty

## How To Write

Prefer clear, warm, inspectable language.

Good Alpha writing should:

- make assumptions visible
- separate decisions from open questions
- avoid hype when care is more useful
- avoid pretending that abstract terms are already solved
- describe impact on people, not only system behavior
- preserve nuance when context changes meaning

## Public Repo Behavior

This repo should be a friendly, serious, curious place.

Do not use it to:

- harass people
- bully people
- promote hate
- promote criminal harm
- sexualize or exploit unsafe situations
- copy work without respect or permission
- force premature certainty

Do use it to:

- learn
- connect
- respect
- amuse
- challenge kindly
- build with care
