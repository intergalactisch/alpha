# Main Implementation

This folder is reserved for a future integrated Alpha implementation.

It should stay empty or minimal until the concept, specs, and protocols are strong enough to justify a main implementation path.

The main implementation should not begin by copying the shape of a normal app.

It should begin from:

- intent
- context
- user control
- agreed outcomes
- transparent process
- memory care
- media-agnostic interaction
- cost-aware execution
- graceful failure
