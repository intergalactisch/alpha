# Protocol Index

Protocols describe how Alpha-shaped parts communicate.

The goal is not to create bureaucracy. The goal is to make intent, context, permissions, process, memory, artifacts, uncertainty, and outcomes legible between systems and people.

## Protocol Principles

Alpha protocols should:

- preserve human control
- make context visible
- carry intent without flattening it
- distinguish suggestion, draft, commitment, and action
- support transparency and askable reasoning
- handle uncertainty without punishment
- avoid hidden persuasion
- avoid unsafe autonomy
- support deletion, correction, revocation, and rollback
- keep children, memory, culture, and privacy protected
- work across media when possible
